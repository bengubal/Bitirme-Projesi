# Bitirme-Projesi
Travmatik beyin hasarı, acil servislerde hızlı müdahale gerektiren kritik bir durumdur. Yoğun iş yükü altında radyolojik incelemeler ciddi zaman kaybı yaratır. Bu proje; bu sınırlamayı aşmak için, beyin BT verilerinden travma bulgularını derin öğrenmeyle otomatik tespit eden yapay zeka tabanlı bir klinik karar destek sistemi sunmaktadır.
